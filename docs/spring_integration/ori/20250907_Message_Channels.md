메시지 채널 (Message Channels)

메시지가 데이터를 캡슐화하는 핵심 역할을 한다면, **메시지 채널(MessageChannel)**은 **메시지 생산자(Producer)와 메시지 소비자(Consumer)를 분리(decouple)**하는 역할을 합니다.
